1-Put all of the program files into one place

2-Open linux console

3-Go to the file where program files are in with cd command, you can use cd .. to go back to previous file

4-After getting into the file use the following command line
	g++ Main.cpp -o fortune.exe

5-After the process is completed you can start the program with ./fortune.exe command

6-You can see the output at output.txt file in where the programs file is in.
